Books Ranking API Documentation

This is A Simple API Documentation for amazon Books Ranking.

provided requests: GET / POST .
The result will return in JSON File.
if the URL is not found you will get Json response url undefined.
in case of an query error response will be with query error.

POST : bookByRank
parameter: rank
book rank parameter is number must be under, equal 10
example with rank:9

parameters: rank, author_rank
book rank + author rank parameter is number must be under, equal 10
